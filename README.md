#  Tecnologias envolvidas

-  ReactJS
-  GraphQL (Apollo Client)
-  Styled Components
-  React Waypont (para Lazy Loading)
-  React Swipeable Routes

#  Rodando o projeto

```
# Clone esse repositorio
git clone ....
 
# Entre no repositorio
cd pokedex-react

# Instale as dependencias
yarn install

# Rode o app
yarn start
```
